Description

A feature-rich task management application that allows users to create, edit, delete, and filter tasks. Includes local storage persistence so tasks survive browser refreshes.
Features

    ✅ Create new tasks

    ✅ Mark tasks as complete/incomplete

    ✅ Edit existing tasks

    ✅ Delete tasks

    ✅ Filter tasks (All/Pending/Completed)

    ✅ Persistent storage with localStorage

    ✅ Smooth animations

    ✅ Responsive design

Technologies Used

    HTML5

    CSS3 (Animations, Flexbox)

    JavaScript (CRUD operations, localStorage)

Learning Outcomes

    CRUD operations

    localStorage API

    JSON parsing/stringifying

    Array methods (filter, map)

    Dynamic UI updates
